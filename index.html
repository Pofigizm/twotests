1. Реализовать функцию asyncForeach и написать для неё юнит-тесты с использованием qunit:
asyncForeach([1, 2, 3], function(item, index, done) {
  setTimeout(function() {
    console.log(item);
    done();
  }, 1000);
}, function() {
  console.log('end');
});

Вывод в консоль:
> 1
> 2
> 3
> "end"

Решение: asyncForeach
